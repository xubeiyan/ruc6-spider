# ruc6-spider
## 介绍
这是爬品知人大的galgame用的爬虫（当然你可以爬任何你想爬的东西），python2.7.10测试通过    
## 用法
安装requests   

`pip install requests`   

更改文件里的用户名和密码   

    unpw = {
      'username':'重写青空after',
      'password':'wwwwwwww', # 这不是我密码
      'quickforward':'yes',
      'handlekey':'ls'
    }

在命令行里执行即可

`python galgame.py`
